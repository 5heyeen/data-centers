## Company Overview

**Metier AS** is Norway's largest project consultancy with over 40 years of experience, ~250 employees, and approximately 800 MNOK in annual revenue. Owned by American Tetra Tech (27,000 employees globally).

- **Headquarters:** Skøyen, Oslo, Norway
- **Purpose:** *"Vi løser utfordringer som betyr noe"* — solving challenges that matter
- **Track record:** 300+ completed projects totalling over 1,000 billion NOK
- **Training:** ~16,000 course participants annually through Metier Academy — the Nordics' largest project school

## Methodologies and Frameworks

| Framework | Domain |
|-----------|--------|
| Metier Project Excellence Framework | Complex physical projects (proprietary) |
| Verdistyrt prosjektutvikling (VPU) | Early-phase value optimization (Target Value Design / Design to Cost) |
| PRINCE2 / PRINCE2 Agile | Core PM methodology; Metier introduced it to Norway |
| PMI / PMBOK | Project management standards |
| AACE / AACEI | Cost engineering; CEP certification |
| P3M3 | Portfolio/programme/project maturity assessments |
| MSP / MoP | Programme and portfolio management |

| LEAN / Agile / Scrum / SAFe | Agile methods for digital and product development |
| Half Double | Hybrid project management methodology |
| KS1, KS2, KSK | Norwegian public quality assurance framework |
| BREEAM | Sustainability certification for buildings |
| PDRI | Project Definition Rating Index |
| Pre-FEED / FEED / EPC | Engineering design and contract phases |
